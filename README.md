# FullStackGuidelines
Full Stack Guidelines for beginners

## Development environment settings
- Download
    1. Python
        ```
        brew install python
        ```
    2. Node / Vue
        1. Install Node
            ```
            brew install node
            ```
        2. Install Vue
            ```
            npm install -g @vue/cli
            ```
    3. PostgreSQL
        1. Install postgreSql
            ```
            brew install postgresql
            ```
        2. Start DB server
            ```
            brew services start postgresql
            ```
